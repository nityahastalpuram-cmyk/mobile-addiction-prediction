# Mobile Addiction Prediction using Machine Learning

## Project Overview
This project analyzes smartphone usage patterns to predict mobile addiction using machine learning techniques.

## Machine Learning Models Used
- Logistic Regression
- Random Forest
- XGBoost

## Dataset
Mobile addiction dataset containing smartphone usage, social media usage, and behavioral indicators.

## Results
The models were compared based on accuracy.

Best Performing Model:
Random Forest – 97.37% Accuracy

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Data Visualization
